🏏 Cricket Auction App
🔒 Confidential Client Project — Going to Production
Overview

A web platform that simulates an IPL-style player auction where multiple teams bid for players in real time, integrated with live match data for fantasy points and automatic leaderboard updates.

The Problem

Private cricket leagues were running auctions manually via spreadsheets and messaging groups — causing bidding confusion, budget tracking errors, and time-consuming fantasy point calculations.

Key Features Built
Real-time player bidding system
Team purse & budget validation
Admin auction control panel
Player transfer system
Player swap between teams
Auto leaderboard updates
Auction queue management
SportsMonks API integration
Technical Architecture

Tournament → Teams → Players → Fixtures → Match Statistics — a clear hierarchy supporting multiple tournament formats like IPL and international events.

Real-time Bidding SportsMonks API Fantasy Points Engine Admin Dashboard Live Leaderboards
Outcome

Replaced 100% manual auction processes with a fully digital platform. Organizers can now run live auctions, manage transfers, and get automatic fantasy rankings based on real match performance.
